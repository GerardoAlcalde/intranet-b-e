# Design Direction

- Tone: editorial, restrained, premium, calm.
- Layout: wide desktop composition, strong hero, balanced side panel, varied section rhythm.
- Typography: `Source Sans 3` for all text, including headlines, body, labels, and navigation.
- Color: warm paper background, deep ink text, subtle neutral borders, one dark accent for interactive elements.
- UI rules: avoid generic SaaS cards, avoid glassmorphism, keep section spacing intentional, keep components small and reusable.
- Navigation: fixed dark top bar with red active state, compact right-side utilities, and a square red B&E logo block.
- Components: shared site chrome lives in the site layout, pages should only render page content.
